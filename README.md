### Framework of App
#### Layout
* Landing
* Authorization
* User Experience :
  * Dashboard
  * Available Movies
  * Selection of Movie
  * Payment Method(s)
  * Video Playback
  * Cast / Film Insights (Overlay)
  * Ratings and Reviews
  * User Profile
  * Settings

#### Features
* **Movies**
   * **Only** movies that are in theaters
   * Highest quality (*video and sound*)
* **Details**
   * Offers insight and details (*overlay*)
* **Ratings System**
   * Ratings from nunu users (*can be focused to target demographic*)
   * 3rd party ratings
